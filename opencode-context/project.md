# Projeto OpenCode

Este é o workspace de configuração do OpenCode.

## Visão Geral
- **Tipo**: Configuração do Sistema
- **Stack**: OpenCode Multi-Agent
- **Objetivo**: Gerenciar configuração e agentes para o projeto ManagerKit

## Agentes Configurados
- `@Maestro` - Orquestrador Principal (não executa, apenas delega)
- `@Artist` - Design UI/UX Premium
- `@Engine` - Backend e Lógica
- `@Guardian` - Segurança, Testes e Diagnóstico
- `@Librarian` - Conhecimento e Contexto
- `@explore` - Exploração somente-leitura da codebase

## Tecnologias
- OpenCode Framework
- Multi-Agent Orchestration
- Skills System
- Supabase (PostgreSQL)
- React 18 + TypeScript + Vite
- TanStack Query
- Shadcn/UI + Tailwind CSS
