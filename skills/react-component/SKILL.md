# React Component Skill

## Detecção de Stack
- Detecta framework do package.json (React, Vue, Svelte, etc)
- Detecta CSS (Tailwind, CSS Modules, styled-components)
- Detecta biblioteca de gráficos (Recharts, Chart.js, etc)
- Detecta UI components (Shadcn, Radix, Headless UI)

## Estrutura de Componente
- Arrow functions com async/await
- Props tipadas com TypeScript
- Componentes funcionais apenas

## Nomenclatura
- PascalCase: Button, Card, Modal
- camelCase: handleClick, fetchData
- Arquivos: button.tsx, use-auth.ts

## Padrões Comuns
- TanStack Query para data fetching
- Context para estado global
- Hooks customizados para lógica reutilizável
